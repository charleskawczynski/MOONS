\section{Parameters to Change for a MOONS Simulation}
Several items must be adjusted in MOONS to correctly model a problem. Below is a short and general list of what must be adjusted and where.

\begin{itemize}
\item Simulation parameters / grid - MOONS.f90
\item Boundary conditions in respective solver folders - initialize\_(NAME)\_BCs.f90
\item Initial conditions in respective solver folders - initialize\_(NAME).f90
\item Input / Output in respective solvers - (NAME)Solver.f90
\item Time varying terms in repective solvers - (NAME)Solver.f90
\item Simulation parameters - simParams.f90
\item Embed / extract options - embedExtract.f90
\end{itemize}
